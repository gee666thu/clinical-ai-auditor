# LearnLens MedPro: Clinical AI Auditor

An enterprise-grade hybrid AI/ML and Generative AI clinical case auditing and forensic data reconciliation engine.

## 🚀 Core Features
* **Live Clinical Telemetry Matrix:** Real-time data processing parsing for patient presentation complexity metrics and case metrics.
* **Forensic EHR Reconciliation Engine:** Local regular expression rule sets scanning medical text for internal physiological boundary anomalies and vital metric verification.
* **Clinical Decision Intelligence Dashboard:** Structured JSON Groq Llama-3.1 cloud orchestrations highlighting overlooked differentials, red flags, and simulation workflows.

## 🛠️ Technology Stack
* **Frontend Dashboard UI:** Streamlit Web Engine Frame
* **Backend Runtime Env:** Python 3 Core Language
* **AI Processing Model Infrastructure:** Groq Cloud SDK Interface (Llama-3.1-8b-instant API)
